一、认证与登录
================================================================================

1.1 APP端登录 (密码登录)
--------------------------------------------------------------------------------
接口地址: POST https://passport2-api.chaoxing.com/v11/loginregister
Content-Type: application/x-www-form-urlencoded

请求参数:
  uname      - 手机号/学号
  code       - 密码 (DES加密后的密文)
  loginType  - 1
  roleSelect - true
  cx_xxt_passport - json

密码DES加密:
  密钥: u2oh6Vu^HWe40fj (Doraemon项目) 或 u2oh6Vu^ (其他项目)
  算法: DES/ECB/PKCS5Padding
  流程: 原始密码 → DES加密 → Hex编码

请求体示例:
  uname=13800138000&code=a1b2c3d4e5f6&loginType=1&roleSelect=true&cx_xxt_passport=json

返回: JSON
  {"status": true, "msg": "验证通过", ...}
  登录成功后从 Set-Cookie 中提取: UID, uf, _d, vc3, _uid, fid, lv 等


1.2 Web端登录 (密码登录)
--------------------------------------------------------------------------------
接口地址: POST http://passport2.chaoxing.com/fanyalogin
Content-Type: application/x-www-form-urlencoded
Header: X-Requested-With: com.chaoxing.mobile

请求参数:
  uname             - 手机号/学号
  password          - DES加密后的密码 (密钥: u2oh6Vu^)
  fid               - 学校ID (默认-1)
  t                 - true
  refer             - https%253A%252F%252Fi.chaoxing.com
  forbidotherlogin  - 0
  validate          - (验证码，一般留空)

请求体示例:
  uname=13800138000&password=a1b2c3d4e5f6&fid=-1&t=true&refer=https%253A%252F%252Fi.chaoxing.com&forbidotherlogin=0&validate=

返回: JSON
  {"status": true, "msg": "验证通过", ...}


1.3 手机验证码登录
--------------------------------------------------------------------------------
发送验证码:
  GET https://passport2-api.chaoxing.com/api/sendcaptcha?phone=13800138000

验证码登录:
  POST https://passport2-api.chaoxing.com/v11/loginregister?cx_xxt_passport=json

  请求参数:
    uname     - 手机号
    code      - 验证码 (同样需要DES加密)
    loginType - 1
    roleSelect - true

  加密内容格式: {"uname":"手机号","code":"验证码"}


1.4 二维码扫码登录
--------------------------------------------------------------------------------
生成二维码:
  GET https://passport2.chaoxing.com/createqr?uuid={随机UUID}

查询扫码状态:
  GET https://passport2.chaoxing.com/getauthstatus?uuid={UUID}

  返回状态:
    0  - 未登录
    1  - 已扫描
    2  - 取消扫描
    3  - 验证通过 (此时返回Cookie)
    4  - 已过期


1.5 登录页面 (移动端)
--------------------------------------------------------------------------------
接口地址: GET http://passport2.chaoxing.com/mlogin?fid=&newversion=true&refer=http%3A%2F%2Fi.chaoxing.com
用途: 获取登录页面HTML

3.1 获取所有课程列表 (JSON)
--------------------------------------------------------------------------------
接口地址: GET http://mooc1-api.chaoxing.com/mycourse/backclazzdata
认证方式: Cookie
可选参数: view=json&mcode=

返回: JSON
{
  "result": 1,
  "msg": "获取成功",
  "channelList": [
    {
      "cataName": "课程",
      "cpi": 197696045,
      "key": 81725723,                    // classId
      "content": {
        "studentcount": 61,
        "name": "班级名称",
        "course": {
          "data": [{
            "id": 236709512,               // courseId
            "name": "课程名称",
            "teacherfactor": "教师姓名",
            "schools": "学校名称",
            "imageurl": "课程封面URL",
            "belongSchoolId": "1840"
          }]
        }
      }
    }
  ]
}

关键字段映射:
  channelList[].key              → classId
  channelList[].cpi              → cpi (后续接口需要)
  channelList[].content.name     → 班级名称
  channelList[].content.course.data[0].id           → courseId
  channelList[].content.course.data[0].name         → 课程名
  channelList[].content.course.data[0].teacherfactor → 教师名
  channelList[].content.course.data[0].schools      → 学校名
  channelList[].content.course.data[0].imageurl     → 封面图

--------------------------------------------------------------------------------

协议: Sec-WebSocket-Protocol: xmpp

连接流程:
  1. 发送: <open xmlns='urn:ietf:params:xml:ns:xmpp-framing' to='cx-dev_cxstudy.easemob.com' version='1.0'/>
  2. 收到: <stream:features> → 发送 SASL 认证
     认证数据: Base64(\0 + myTuid + \0 + myToken)
     发送: <auth xmlns='urn:ietf:params:xml:ns:xmpp-sasl' mechanism='urn:xmpp:oauth2:0'>{Base64}</auth>
  3. 收到: <success> → 绑定资源
     发送: <iq type='set' id='_bind_auth_2'><bind xmlns='urn:ietf:params:xml:ns:xmpp-bind'><resource>webim_{timestamp}</resource></bind></iq>
  4. 收到: <bind result> → 建立会话
     发送: <iq type='set' id='_session_auth_2'><session xmlns='urn:ietf:params:xml:ns:xmpp-session'/></iq>
  5. 上线: <presence xmlns='jabber:client'><priority>1</priority></presence>
  6. 收到: <message> → 解析消息

环信配置:
  AppKey:   cx-dev#cxstudy
  XMPP域:   easemob.com
  API地址:  https://a1-vip6.easecdn.com

附录A: 完整签到流程
================================================================================

1. 登录获取Cookie
   POST passport2-api.chaoxing.com/v11/loginregister

2. 获取课程列表
   GET mooc1-api.chaoxing.com/mycourse/backclazzdata
   → 提取 courseId, classId, cpi

3. 获取活动列表
   GET mobilelearn.chaoxing.com/ppt/activeAPI/taskactivelist
   → 找到 activeType=2 且 status=1 的活动
   → 提取 activeId (aid), preSignUrl

4. 获取签到类型
   GET mobilelearn.chaoxing.com/newsign/signDetail?activePrimaryId={aid}&type=1
   → 根据 otherId 判断签到类型

5. 预签到
   GET mobilelearn.chaoxing.com/newsign/preSign?courseId=...&classId=...&activePrimaryId=...

6. 签到前置分析 (部分类型需要)
   GET mobilelearn.chaoxing.com/pptSign/analysis?aid={aid}
   GET mobilelearn.chaoxing.com/pptSign/analysis2?code={code}

7. 执行签到
   根据签到类型调用对应接口:
   - 普通签到: pptSign/stuSignajax?activeId={aid}&uid={uid}&...
   - 位置签到: pptSign/stuSignajax?address=...&latitude=...&longitude=...
   - 拍照签到: pptSign/stuSignajax?objectId={objectId}&...
   - 二维码签到: pptSign/stuSignajax?enc={enc}&...
   - 签到码: 先checkSignCode验证，再pcStuSignController/signIn

8. 验证结果
   返回 "success" → 签到成功


附录B: 自动监听签到流程 (IM方式)
================================================================================

1. 获取IM参数
   GET im.chaoxing.com/webim/me → myToken, myTuid

2. 连接WebSocket
   wss://im-api-vip6-v2.easecdn.com/ws
   → XMPP认证流程

3. 监听消息
   收到 <message> → 解析 ext.attachment.att_chat_course
   → url包含"sign" → 提取 aid, classId, courseId

4. 自动签到
   获取签到类型 → 预签到 → 执行签到
